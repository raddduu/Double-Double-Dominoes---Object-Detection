2. Rularea task-urilor

Pe a doua linie puneti calea(pornind din root-ul proiectului) catre folderul in care doriti sa se puna fisierele text rezultate
Includeti folderul de "templates" in root directory
Rulati solution.py. Rezultatele vor fi in fisierul specificat. Pentru un singur meci ar trebui sa dureze rularea maxim 1-2 minute
